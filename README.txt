2. This repository will include the progress of the following tasks.
- Conduct research on reinforcement learning algorithms, with a focus on average-reward and risk-sensitive settings
- Investigate risk-aware optimization techniques for minimizing catastrophic outcomes in sequential decision-making systems
- Perform experimental analysis, including benchmarking algorithms, tuning hyperparameters, and evaluating performance metrics 
- Develop and implement machine learning models using frameworks such as PyTorch 
- Develop and implement a deep learning model CNN using frameworks such as PyTorch or TensorFlow
- Design and evaluate reinforcement learning–augmented small-scale transformer models, using reward shaping and policy optimization (e.g., PPO) to study alignment, decision-making behavior, and performance-efficiency trade-offs under limited computational resources.
- Document research findings through technical reports, code documentation, and reproducible experiments
